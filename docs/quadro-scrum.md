# Quadro Scrum

## Ferramenta Utilizada

**Platarforma:** Trello

**Link do público do quadro:** https://trello.com/invite/b/69a034aba1a5ffd096de0a01/ATTI7349f454607e64dad0689d88e108260aED6CD471/sistemas-de-receitas-culinarias

## Estrutura do Quadro

O quadro contém as seguintes colunas:

1. **Backlog do Produto:** - Ideias e funcionalidades desejadas.
2. **Sprint Backlog:** - Itens escolhidos para a sprint atual.
3. **Em andamento:** - Tarefas que estão sendo executadas.
4. **Em revisão** - Tarefas concluídas aguardando validação.
5. **Cloncluido:** - Tarefas totalmente finalizadas.

## Regras de trabalho

- **Duração da sprint:** 1 semana
- **Reunião semanal:** Média (40-60min) - para atribuir tarefas aos integrantes e discutir sobre o desenvolvimento do projeto.
- **Ferramentas de comunicação:** WhatsApp + Trello + GitHub

## Definição de "Concluído"

- Código testado e funcionando.
- Revisado por outro membro.
- Documentado no README.
- Cartão movido para "concluído".
